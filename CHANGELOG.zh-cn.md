# 更新日志

"cc-ring" 扩展的所有重要更改都将记录在此文件中。

查看 [Keep a Changelog](http://keepachangelog.com/) 了解如何组织此文件的建议。

## [0.2.1] - 2025-11-12

### 修复
- 修复中文本地化字符串中的引号转义问题

## [0.2.0] - 2025-11-11

### 新增
- 简体中文本地化支持（UI 和市场）
  - 所有扩展 UI 字符串现在支持中文翻译
  - 中文自述文件（README.zh-cn.md）
  - 中文更新日志（CHANGELOG.zh-cn.md）
  - 根据 VS Code 语言设置自动检测语言
- 使用 VS Code 现代 l10n API 的本地化基础设施
  - l10n/ 目录中的英文和中文包文件
  - 用于市场本地化的 package.nls.json 文件

## [0.1.1] - 2025-11-11

### 修复
- 改进了对损坏的 settings.json 文件的错误处理
- 更好地处理重复的钩子安装
- 在安装/卸载过程中正确保留现有的 Claude Code 钩子
- 使用一致的 UUID 修复了钩子识别问题
- 增强了错误消息，适当区分警告和错误分类
- 稳健处理 settings.json 中格式错误的钩子结构

### 新增
- 综合测试套件（42 个通过的测试）

## [0.1.0] - 2025-11-11

### 新增
- CC Ring 扩展首次发布
- Claude Code 完成任务时的声音通知
- 可配置的音量控制（0-100%）
- 多个内置声音选项：
  - 完成：标准完成提示音
  - 细微：安静的细微提示音
  - 通知：警报通知提示音
- 自定义声音文件支持
- 命令：
  - 测试通知声音：预览所选声音
  - 选择自定义声音文件：选择自定义声音文件
  - 安装/重新安装钩子：设置 Claude Code 集成
  - 卸载钩子：移除 Claude Code 集成
  - 显示状态：显示当前钩子安装状态
- 配置设置：
  - 启用/禁用通知
  - 音量控制
  - 声音选择
  - 自定义声音文件路径
- Claude Code 集成的自动钩子管理
